# 論文サーベイスキル

## 概要
SDGs・環境政策・子ども政策に関する学術文献を体系的に調査・整理するスキルです。
複数のMCPツールを組み合わせて網羅的な文献調査を行います。

## 使用するMCPツール

### 論文検索
- `paper-search`: 主要学術データベース横断検索（26ソース）
- `arxiv`: プレプリント・新興研究の探索
- `semantic-scholar`: 引用ネットワーク分析・著者検索

### Web補完
- `google-search`: 日本語の政策文書・政府レポート
- `brave-search`: 英語の国際機関レポート・NGO資料
- `fetch`: 特定文書の全文取得

## サーベイプロセス

### フェーズ1: スコープ設定
```
1. キーワードの整理
   - メインキーワード（例: SDGs governance）
   - 関連キーワード（例: sustainable development, multi-stakeholder）
   - 除外キーワード（例: SDGs marketing [マーケ文脈を除外]）

2. 期間の設定
   - 通常: 過去10年（2015年以降）
   - SDGs関連: 2015年（採択年）以降
   - レビュー論文は年限を外す場合もあり

3. 言語の設定
   - 英語（国際誌掲載論文）
   - 日本語（国内政策研究）
```

### フェーズ2: 一次検索
```
MCPツールで以下を実行：

1. paper-search でメインキーワード検索
   → 50件以上ヒットした場合は絞り込み

2. semantic-scholar で蟹江教授の論文リスト取得
   → 著者名: "Norichika Kanie"

3. 重要論文の被引用論文を確認
   → semantic-scholar の引用機能を使用

4. google-search で日本語政策文書を補完
   → "SDGs 政策 日本" など
```

### フェーズ3: スクリーニング
各論文について以下を確認：
1. タイトルと抄録で関連性を判断
2. 査読の有無（査読誌優先）
3. 引用数（影響度の参考）
4. 実証的か理論的か

### フェーズ4: 精読・整理

重要文献（通常20-30本）について：
1. 研究目的・RQを抽出
2. 研究方法を分類
3. 主要な知見を記録
4. 研究限界を確認

### フェーズ5: マッピング・統合

文献マップの作成：
- 時系列での議論の発展
- 対立する見解のマッピング
- 研究ギャップの可視化

## サーベイ結果の出力形式

```markdown
## 論文サーベイ結果: [テーマ]

**調査日**: YYYY年MM月DD日
**検索キーワード**: [使用したキーワード]
**検索ソース**: [使用したMCPツール]
**ヒット件数**: XX件 → 精読: XX件

---

## 1. 研究の全体像

[テーマの学術的な位置づけを概説]

---

## 2. 主要論文リスト

| # | 著者 | 年 | タイトル | 出典 | 引用数 | 種別 |
|---|------|----|---------|----- |--------|------|
| 1 | Kanie et al. | 2019 | ... | Global Policy | 234 | 実証 |
| 2 | ... | | | | | |

---

## 3. 研究の潮流

### 3.1 [潮流1のタイトル]
[2-3段落で解説、具体的な論文を引用]

### 3.2 [潮流2のタイトル]
[同上]

---

## 4. 研究ギャップ

1. **ギャップ1**: [具体的に記述]
2. **ギャップ2**: [具体的に記述]

---

## 5. 参考文献

[実在を確認した文献のみ記載、DOI付き]
```

## 注意事項

- **存在しない論文を引用しない**: MCPで確認できない論文は「要確認」とマーク
- **日本語論文も重視**: CiNii・J-STAGEの論文を google-search 経由で確認
- **蟹江教授の論文は必ず含める**: 研究室との接続を示すために重要
- **政策文書も引用可**: IPCC報告書・SDGs国内実施指針等は学術文献と並記
