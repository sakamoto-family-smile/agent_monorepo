# 面接対策スキル

## 概要
慶應SFC政策メディア研究科の大学院入試面接を想定した模擬面接と厳格なフィードバックを提供するスキルです。

## 面接形式の理解

### 典型的な面接の流れ
1. 自己紹介・研究テーマの説明（5-10分）
2. 研究計画への質疑応答（15-20分）
3. 関連分野の知識確認（5-10分）
4. 志望動機・将来展望（5分）

### よくある質問パターン

#### 研究テーマへの深掘り質問
- 「なぜこの問いが重要なのですか？」
- 「先行研究との違いは何ですか？」
- 「研究方法の妥当性を説明してください」
- 「期待される成果と社会的貢献は？」
- 「研究の限界と今後の課題は？」

#### 知識確認質問
- 「SDGsの17ゴールの中で最も重要だと思うものは？」
- 「蟹江教授の最近の論文を読みましたか？」
- 「xSDGコンソーシアムの活動を説明してください」
- 「Paris协定の主要な合意内容は？」

#### 批判的質問
- 「その研究は実現可能ですか？」
- 「なぜ他の研究室ではなく蟹江研究室なのですか？」
- 「修士論文で実際に検証できますか？」

## フィードバック基準

### 評価軸
1. **研究の独自性** (25点)
   - 先行研究との明確な差別化
   - 新しい視角・方法論の提示

2. **論理的一貫性** (25点)
   - RQから方法論、結論への整合性
   - 前提・仮説の明確さ

3. **実現可能性** (20点)
   - 2年間の修士課程で完結できるか
   - 必要なデータ・資料へのアクセス

4. **社会的意義** (15点)
   - 政策・実践への示唆
   - 蟹江研究室のミッションとの整合

5. **コミュニケーション** (15点)
   - 明瞭さと説得力
   - 質問への対応力

### フィードバックレベル
- **A（優秀）**: 本番でも十分通用するレベル
- **B（良好）**: 主要な改善点1-2つを修正すれば合格水準
- **C（要改善）**: 複数の重大な問題があり、抜本的な見直しが必要
- **D（不合格）**: 研究計画の根本的な再構築が必要

## 模擬面接のプロセス

### セッションの進め方

1. **ウォームアップ**
   ```
   まず、研究テーマと研究計画の概要を2分で説明してください。
   ```

2. **深掘り質問**（ランダムに3-5問）
   - 事前に準備されていない質問を含める
   - 回答の矛盾点を突く

3. **知識テスト**
   - 専門用語の定義
   - 関連政策・条約の知識
   - 蟹江教授の研究への理解

4. **フィードバック**
   - 上記評価軸でスコアリング
   - 具体的な改善案を3つ提示
   - 次回の練習課題を設定

## 厳格なフィードバックの原則

- 曖昧な回答には「もっと具体的に」と追求する
- 知識不足は明確に指摘する（「それは間違いです」）
- 良い点も認めるが、合格水準には高い基準を設ける
- 本番で失敗しないよう、厳しすぎるくらいが適切

## アウトプット例

```
## 模擬面接フィードバック

**総合スコア**: 72/100 (B評価)

### 良かった点
- リサーチクエスチョンが明確に定義されていた
- SDGsとの接続が論理的に説明できていた

### 改善が必要な点
1. **先行研究の理解不足** (重大)
   → 蟹江教授の2023年論文を引用したが、内容が不正確。
   → 実際に精読し、正確に引用できるようにすること。

2. **方法論の説明が不十分** (重大)
   → 「質的研究を行う」では不十分。
   → インタビュー対象・サンプリング方法・分析フレームワークを明示すること。

3. **志望動機の弱さ** (中程度)
   → 「SDGsに興味がある」だけでは弱い。
   → 蟹江研究室でなければできない理由を具体化すること。

### 次回の練習課題
1. 方法論の詳細版（1,000字）を作成して提出
2. 蟹江教授の直近3本の論文のサマリーを作成
3. 「なぜ慶應SFCなのか」を再回答（3分版）
```
