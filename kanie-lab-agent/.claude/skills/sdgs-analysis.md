## 分析フレームワーク

### 1. SDGsガバナンス分析

#### アクター分析
- **国家アクター**: 各国政府・地方自治体
- **国際機関**: UN, UNDP, UNESCO等
- **民間セクター**: 企業・産業界（xSDGコンソーシアム含む）
- **市民社会**: NGO/NPO・学術機関
- **個人**: 市民・地域住民

#### メカニズム分析
- **ハードガバナンス**: 条約・法規制・強制的措置
- **ソフトガバナンス**: ガイドライン・自発的コミットメント
- **実験的ガバナンス**: パイロット事業・イノベーション

### 2. 政策統合分析（Policy Coherence for SDGs）

政策の水平的・垂直的統合を評価：
```
水平的統合: 省庁間・セクター間の整合性
垂直的統合: 国際-国家-地方の整合性
時間的統合: 短期目標と長期目標の整合性
```

### 3. トレードオフ・シナジー分析

SDGsゴール間の相互関係：
- **シナジー**: 一つのゴール達成が他のゴールを促進
- **トレードオフ**: 一つのゴール達成が他のゴールと競合
- **SDGs Compass**: 企業向けSDGs統合ツール

### 4. インパクト評価

VLR（自発的地方レビュー）・VNR（自発的国家レビュー）の分析：
- 目標設定の妥当性
- 測定指標の適切性
- 進捗報告の透明性

## 主要な研究視点

### xSDGsアプローチ（蟹江研究室の特徴）
xSDGs = SDGsを実装・変革するための具体的アプローチ：
- トランスフォーメーション（社会変革）の視点
- 科学と政策のインターフェース
- プライベートセクターの役割

### 2030アジェンダ後の議論
2030年以降のSDGs枠組み（Post-2030 Agenda）：
- SDGsの達成状況評価
- 新たな課題（AI・デジタル等）の統合
- 実施手段（ファイナンス）の強化

## 調査手順

```
1. google-search で最新の国内SDGsレポートを確認
   → 「SDGsアクションプラン」「VLR」等

2. brave-search で国際的なSDGsレポートを確認
   → "HLPF" "VNR" "SDG progress report"

3. paper-search でSDGsガバナンス論文を検索
   → "SDG governance" "SDG implementation"

4. semantic-scholar で蟹江教授の関連論文を確認
   → 著者: Norichika Kanie

5. fetch で特定のレポート全文を取得
   → 国連・政府の公式文書URL
```
