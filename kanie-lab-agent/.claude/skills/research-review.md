# 研究計画レビュースキル

## 概要
大学院入試用の研究計画書を7軸評価し、具体的な改善案を提示するスキルです。
慶應SFC政策メディア研究科の選考基準を考慮した厳格な評価を行います。

## 7軸評価フレームワーク

### 軸1: 問題の明確性（20点）
研究が解決しようとする問題の明確さと重要性を評価。

**評価基準**:
- A（18-20点）: 問題が明確に定義され、社会的重要性が十分に論証されている
- B（14-17点）: 問題は理解できるが、一部の論拠が不明確
- C（10-13点）: 問題定義に曖昧さがあり、重要性の根拠が弱い
- D（0-9点）: 問題の定義が不明確か、重要性が示されていない

**評価チェックリスト**:
- [ ] 解決しようとする問題が一文で説明できるか
- [ ] 問題の規模・影響を示す数値的根拠はあるか
- [ ] なぜ今この問題に取り組む必要があるかが説明されているか
- [ ] 問題の受益者（誰が困っているか）が明確か

### 軸2: リサーチクエスチョンの精度（15点）
研究の中心的な問いの明確さと研究可能性を評価。

**評価基準**:
- A（14-15点）: RQが明確・具体的で、研究方法と対応している
- B（11-13点）: RQは理解できるが、やや広すぎるか曖昧な表現がある
- C（8-10点）: RQが漠然としており、実証研究への落とし込みが困難
- D（0-7点）: RQが存在しないか、研究不可能な問いになっている

**評価チェックリスト**:
- [ ] RQは1-2文で明確に表現されているか
- [ ] RQに答えることで何が明らかになるか分かるか
- [ ] RQは2年間の修士課程で研究可能な範囲か
- [ ] RQは「なぜ・どのように」を問う分析的なものか

### 軸3: 先行研究レビューの質（20点）
先行研究の理解と研究ギャップの特定を評価。

**評価基準**:
- A（18-20点）: 主要先行研究を網羅し、ギャップを明確に示している
- B（14-17点）: 重要な先行研究は網羅しているが、ギャップの説明が不十分
- C（10-13点）: 先行研究の引用が不足しているか、ギャップが不明確
- D（0-9点）: 先行研究の理解が著しく不足している

**評価チェックリスト**:
- [ ] 主要な英語論文（査読誌）が5本以上引用されているか
- [ ] 蟹江教授の関連論文が適切に引用されているか
- [ ] 先行研究の限界（ギャップ）が具体的に示されているか
- [ ] 引用した論文が実在するか（MCPで確認済みか）

### 軸4: 研究方法の妥当性（20点）
研究方法の選択とその妥当性を評価。

**評価基準**:
- A（18-20点）: 方法論が明確で、RQと整合し、実施可能性が高い
- B（14-17点）: 方法は適切だが、具体性やRQとの整合性に一部問題がある
- C（10-13点）: 方法論が不明確か、RQに対して不適切な方法が選択されている
- D（0-9点）: 方法論の記述がほぼなく、研究の実施可能性が不明

**評価チェックリスト**:
- [ ] 定量・定性・混合のどれを使うか明示されているか
- [ ] データ収集方法（インタビュー・アンケート・2次データ等）が具体的か
- [ ] サンプリング方法・対象・規模が示されているか
- [ ] 分析手法（内容分析・統計分析等）が明示されているか

### 軸5: 実現可能性（10点）
2年間の修士課程での完成可能性を評価。

**評価基準**:
- A（9-10点）: タイムラインが合理的で、必要なリソースへのアクセスが可能
- B（7-8点）: 概ね実現可能だが、一部にリスクがある
- C（5-6点）: 実現に大きな障壁がある可能性が高い
- D（0-4点）: 現実的に完成が困難な計画

**評価チェックリスト**:
- [ ] 研究フェーズと大まかなスケジュールが示されているか
- [ ] 必要なデータへのアクセス見通しはあるか
- [ ] インタビュー対象者へのアクセスは現実的か
- [ ] 修士論文として完結する範囲に絞られているか

### 軸6: 蟹江研究室との適合性（10点）
研究室の専門領域・活動との接続性を評価。

**評価基準**:
- A（9-10点）: 研究室の専門領域・活動と深く接続しており、相乗効果が期待できる
- B（7-8点）: 研究室との接続はあるが、もう少し具体的な説明が必要
- C（5-6点）: 研究室との接続が表面的または説明が不十分
- D（0-4点）: 研究室の専門領域と乖離している

**評価チェックリスト**:
- [ ] SDGs・環境政策・ガバナンスとの明確な接続があるか
- [ ] xSDGコンソーシアムや蟹江教授の活動との関連性があるか
- [ ] 蟹江研究室でなければできない理由が説明できるか
- [ ] SFCの学際的・実践的な研究スタイルと合っているか

### 軸7: 社会的貢献・政策インパクト（5点）
研究成果の実践的な貢献可能性を評価。

**評価基準**:
- A（5点）: 具体的な政策・実践への貢献が明確に示されている
- B（4点）: 貢献の方向性は示されているが、具体性が不足
- C（3点）: 社会貢献への言及があるが、漠然としている
- D（0-2点）: 学術的貢献のみで実践的示唆が示されていない

```markdown
## 詳細フィードバック

### 軸2: リサーチクエスチョン（要改善）

**問題点**:
現在のRQは「SDGsガバナンスはどのように機能するか」と広すぎる。
2年間で答えられる具体性がなく、何を明らかにするかが不明確。

**改善案**:
「日本の地方自治体によるSDGsローカライゼーションにおいて、
マルチステークホルダー・パートナーシップはどのような条件下で
効果的に機能するか」のように、対象・変数・文脈を明確にする。

### 軸4: 研究方法の妥当性（要改善）

**問題点**:
「事例研究を行う」とのみ記述されており、方法論の詳細がない。
どの自治体を対象に、何件のインタビューを、どう分析するかが不明。

**改善案**:
- 対象自治体: SDGsモデル都市3-5市町村
- インタビュー対象: 担当職員・企業代表・NPO代表（各2-3名）
- 分析手法: プロセストレーシング + Nvivo による質的分析

---

## 改善版の提案

## 次のアクション（優先順）

2. **重要**: 方法論の詳細（サンプリング・分析手法）を追記
3. **推奨**: 蟹江教授の2022-2024年の論文を精読し、引用を追加
```

## レビュー時の注意事項

- 存在しない論文への言及は明確に指摘する
- 実現不可能な研究計画には率直にフィードバックする
- 良い点を認めつつも、合格水準への具体的な改善策を示す
- 必要に応じてMCPで引用文献の存在確認を行う
