# 研究テーマ設計スキル

## 概要
蟹江研究室（慶應SFC）への大学院入試に向けた研究テーマ設計を支援するスキルです。

## 研究テーマ設計プロセス

### ステップ1: 問題意識の明確化
ユーザーの関心領域から以下を抽出する：
- 社会課題（どのような問題を解決したいか）
- 地理的スコープ（グローバル・国内・地域）
- アクターへの関心（政府・企業・市民社会・国際機関）

### ステップ2: 先行研究の確認
MCPツールを使って以下を調査する：
```
1. paper-search で関連論文を検索
2. semantic-scholar で蟹江教授の関連研究を確認
3. google-search で日本の政策動向を確認
4. brave-search で国際的な議論を確認
```

### ステップ3: 研究ギャップの特定
先行研究を整理し、以下の観点でギャップを探す：
- 未解明の問題（What is unknown?）
- 方法論の限界（Methodological gap）
- 地理的・時間的ギャップ（Context gap）
- 実践的示唆の不足（Practice gap）

### ステップ4: リサーチクエスチョンの精緻化
RQは以下の要素を含むこと：
- 明確な従属変数（何を説明するか）
- 独立変数・メカニズム（何が影響するか）
- 文脈・スコープの限定

### ステップ5: 蟹江研究室との接続
以下の軸で研究室との親和性を確認する：
- SDGsガバナンスとの接続
- 政策プロセス分析の要素
- 実証的アプローチの可能性
- xSDGコンソーシアムとの連携可能性

## 推奨テーマ領域

### SDGs・持続可能性ガバナンス
- SDGsの実施メカニズム分析
- マルチステークホルダー・パートナーシップ
- 自治体のSDGsローカライゼーション
- 民間セクターのSDGsコミットメント

### 環境政策
- 気候変動適応政策の比較分析
- カーボンプライシングの政治経済学
- 再生可能エネルギー政策の制度設計
- 生物多様性政策のガバナンス

### 子ども・人口政策
- 少子化対策の政策効果分析
- 子どもの貧困と社会的包摂
- 教育政策とSDGs目標4
- ジェンダー平等政策の比較研究

## アウトプット形式

研究テーマ提案は以下の構造で出力する：

```
## 研究テーマ案

**タイトル（仮）**: [研究タイトル]

**リサーチクエスチョン**: [具体的なRQ]

**研究の背景と意義**:
[社会的背景と学術的意義を2-3段落で記述]

**先行研究と研究ギャップ**:
[主要先行研究3-5本を引用し、ギャップを明示]

**研究方法の案**:
[定量・定性・混合手法の検討]

**蟹江研究室との接続点**:
[研究室の専門領域との具体的な接点]

**参考文献**:
[MCPで確認した実在文献のみ記載]
```
