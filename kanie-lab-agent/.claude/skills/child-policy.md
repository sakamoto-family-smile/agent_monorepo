# 子ども政策スキル

## 概要
子ども政策・少子化対策・子育て支援に関する政策研究を支援するスキルです。
日本の政策分析と国際比較研究に対応します。

## 主要な政策領域

### 1. 少子化対策・出生率政策
- 合計特殊出生率の推移分析
- 少子化対策の効果検証
- 各国の子育て支援政策比較

### 2. 子どもの貧困・格差
- 子どもの貧困率指標（相対的貧困率）
- 教育機会の不平等
- ひとり親世帯への支援

### 3. 児童福祉・保護
- 保育・幼児教育の整備
- 虐待防止政策
- 社会的養護（里親・施設）

### 4. 教育政策
- SDGs目標4（質の高い教育）との接続
- インクルーシブ教育
- 非認知能力の育成

## 主要な統計データ源

### 国内
- **e-Stat**（estat MCPツールで検索可能）
  - 人口動態統計（出生率・死亡率）
  - 国民生活基礎調査
  - 社会保障・人口問題研究所データ

- **子ども家庭庁**
  - 子どもの貧困に関する指標
  - 保育所関係状況

- **内閣府**
  - 少子化社会対策白書
  - 子供・若者白書

### 国際
- **UNICEF** - 子どもの権利・指標データ
- **OECD** - Family Database
- **世界銀行** - 子ども関連指標

## 政策分析フレームワーク

### 政策効果分析
```
1. 目標の明確性
   - 数値目標の設定有無
   - 測定可能な指標の存在

2. 実施メカニズム
   - 予算規模と配分
   - 行政実施体制
   - 民間との連携

3. 評価・改善サイクル
   - PDCAサイクルの機能
   - エビデンスに基づく政策改善

4. アクセスと公平性
   - 支援へのアクセスの平等性
   - 地域格差の存在
```

### 国際比較の視点
北欧モデル・フランスモデルとの比較：
- 育児休業制度の充実度
- 保育サービスの公的負担
- 現金給付 vs. 現物給付

## 調査手順

```
1. estat で国内統計データを検索
   → 人口動態・子どもの貧困率等

2. google-search で最新の政策動向を確認
   → 「子ども家庭庁」「こども大綱」等

3. e-gov-law で関連法令を確認
   → 「子どもの貧困対策の推進に関する法律」等

4. brave-search で国際比較研究を確認
   → "child poverty policy comparison"

5. paper-search で学術研究を検索
   → "child welfare Japan" "少子化 政策"
```

## 蟹江研究室との接続点

- SDGs目標1（貧困）・目標4（教育）・目標10（不平等）との接続

- 政策立案プロセスへの市民参加
- 地方自治体によるSDGsローカライゼーションとしての子ども政策
