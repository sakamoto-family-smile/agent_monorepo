# Cooking Rules

ホットクックの仕様上の制約と、ユーザーに伝えるべき注意点まとめ。
正本は KN-HW24H 取扱説明書。本ドキュメントは agent が参照しやすいよう要点を抜粋。

## 予約調理 NG な食材 / 状況

予約調理は最大 15 時間先まで完成時刻を指定できるが、以下は腐敗・食中毒リスクがあるため
**必ず即時調理を案内する**:

- 生の鶏肉 + 室温時間が長い (夏場は特に注意)
- 生卵 (殻付き / 割った状態とも)
- 牛乳 / 生クリームを最初から入れるレシピ (途中追加なら可)
- 海鮮全般 (鮭・さば・たら・あさり等) — 当日購入即セット以外は不可
- 生のひき肉

> **予約調理 OK な代表例**: 肉じゃが / 筑前煮 / 豚汁 / カレー (鶏肉は冷凍を前提とする等)

## まぜ技ユニットを必ず使うメニュー

`mixer_required=true` のメニューは **まぜ技ユニットの装着確認** を毎回案内する。
装着忘れ → 焦げ・煮崩れ・食材偏りの主原因。

代表例:
- カレー / シチュー全般
- 肉じゃが
- ミネストローネ等の野菜たっぷりスープ
- ミートソース・カルボナーラ等のパスタソース
- ラタトゥイユ
- リゾット

## まぜ技ユニットを使わない方が良いメニュー

- 角煮 / おでん (形を保ちたい)
- 蒸し料理 (鶏ハム / 蒸し野菜 / 茶碗蒸し / シュウマイ)
- 低温調理 (ローストビーフ等)
- 発酵 (ヨーグルト / 甘酒)

## 容量 (KN-HW24H = 2.4L) の上限

- 食材総量 (調味料込み) は **1.6L 程度まで** が目安 (取扱説明書参照)
- パスタ調理は 2 人前 (乾麺 200g) まで
- 米は最大 2合
- ヨーグルトは牛乳パック 1L そのまま入れて作れる

## 焦げ付き / 機能上の制約

- 砂糖を多用するレシピ (照り焼き・甘辛煮) は焦げやすい → 必ず指定通り無水 or 少水で
- 内釜は強アルカリ洗剤 NG (フッ素加工劣化)
- 蒸し料理は内釜に水を 200ml 以上入れる
