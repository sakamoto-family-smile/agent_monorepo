## 1. 道路交通法・施行令・施行規則（e-Gov 法令検索）

### 概要

| 項目 | 内容 |
|---|---|
| 提供元 | デジタル庁（e-Gov 法令検索） |
| 取得手段 | e-Gov 法令検索 API（v2） |
| データ形式 | XML（標準法令 XML スキーマ） |
| 更新頻度 | 法令改正のタイミング（不定期、月次取得で十分） |

### 利用条件（要 Phase 0 再確認）

- e-Gov 法令検索 API の利用規約を確認する。基本的には政府提供のオープンデータとして広く利用可能と見られるが、商用利用条件・出典表記義務・レート制限を必ず一次ソース（e-Gov 法令検索の公式利用規約ページ）で最新確認する。
- 出典表記の例: 「e-Gov 法令検索（〇〇省 / 内閣官房）」相当の記載を解説等の根拠リンク部に併記。

### 取得対象

- 道路交通法（昭和35年法律第105号）
- 道路交通法施行令
- 道路交通法施行規則
- 道路標識、区画線及び道路標示に関する命令

### 運用方針

- 月次バッチ（[DESIGN.md §7](./DESIGN.md#7-法令改正追従パイプライン月次)）で全文を取得し GCS に snapshot を保存
- 前回 snapshot との diff を取り、変更条文に紐づく問題を `needs_review` にフラグ
- 問題スキーマの `sources[].url` には e-Gov の **条文単位アンカー付き URL** を格納（ハルシネーション検証の根拠）

## 2. 交通の方法に関する教則（警察庁）

### 概要

| 項目 | 内容 |
|---|---|
| 提供元 | 警察庁 |
| 取得手段 | 警察庁ウェブサイト掲載の PDF |
| データ形式 | PDF（章節構造あり） |
| 更新頻度 | 不定期 |

### 利用条件（要 Phase 0 再確認）

- 政府機関が公表する著作物は原則「政府標準利用規約 2.0」（CC BY 4.0 互換）の範囲内で利用可能とされるケースが多い。**警察庁発行物が同規約の対象範囲か、各 PDF の利用条件表示を確認すること**。
- 出典表記: 「警察庁『交通の方法に関する教則』（最終確認日 YYYY-MM-DD）」相当を解説の根拠リンク部に併記。
- 直接の引用（条項単位のテキスト引用）は著作権法 32 条の引用要件（公表されたもの・引用の必然性・主従関係・明瞭区別・出典明示）を満たす範囲で行う。

### 構造化方針

PDF を直接 LLM に渡すと精度・コストが悪いため、Phase 0 で章節構造ありの JSON に変換する。

```
kyousoku-corpus.json
  - chapters:
    - chapter: 1
      title: "交通の方法 — 歩行者の心得"
      sections:
        - section: 1
          page: 5
          paragraphs:
            - id: "1-1-1"
              text: "..."
              page: 5
```

- OCR ではなく PDF テキスト抽出 + 手動レビュー（運営者 1 人）で構造化
- 章節 ID と PDF ページ番号を保持し、解説からのリンクは `https://.../kyousoku.pdf#page=NN` で生成

## 3. 道路標識・標示・信号機

### 概要

道路標識、区画線及び道路標示に関する命令で定められた様式に基づく標識データ。

### 取得・調達方針（要 Phase 0 確定）

| 案 | 内容 | メリット | デメリット |
|---|---|---|---|
| **a. 自前 SVG 化** | 公式様式を参照しながら運営者が SVG を作成 | 権利関係クリア、軽量、再利用容易 | 初期工数が大きい |
| **b. 政府オープンデータ** | 国土交通省 / 警察庁等が公開している標識データセットを利用 | 信頼性が高く工数低 | データセットの存在・利用条件は要確認 |
| **c. CC0 / Public Domain の標識集** | Wikimedia Commons 等の Public Domain 画像 | 無料・即利用可 | ライセンス確認必須、デザイン差異の可能性 |

**Phase 0 で b → c → a の順に調査し、揃わない部分のみ a で補完する方針**。

### 標識カタログ JSON

```
signs-catalog.json
  - id: "C-301"          # 規制標識 / 一時停止
    title: "一時停止"
    category: "regulatory"
    image_uri: "gs://.../signs/C-301.svg"
    description: "..."
    legal_basis:
      - law_id: "335M50000003003"
        article_or_form: "別表第二"
```

## 6. データ取得バッチ全体像（Phase 4+）

```
Cloud Scheduler (月初)
  ├─ e-Gov API → GCS: law-snapshots/YYYY-MM/*.xml
  ├─ 警察庁教則 PDF → 手動 or 半自動更新（変更検知 + 運営者通知）
  └─ 標識カタログ → 変更稀少のため手動更新
        │
        ▼
  diff 検出 → 影響問題を needs_review にフラグ
```
