# プライバシーポリシー（初版 / Draft）

- **適用日**: 2026-XX-XX（サービス公開時に確定）
- **バージョン**: 0.1（Phase 0 初版・公開前 draft）
- **発行者**: 個人運営（運営者氏名は公開時に確定）

> 本ポリシーは driving-license-bot（以下「本サービス」）における個人情報の取扱いについて定めるものです。

---

## 1. 取得する情報

本サービスは、以下の情報を取得します。

### 1.1 LINE プラットフォーム経由で自動的に取得する情報

| 情報 | 取得タイミング | 用途 |
|---|---|---|
| LINE User ID（U で始まる 33 文字） | 友だち追加・メッセージ送信時 | ユーザー識別、内部 UUID とのマッピング |
| 表示名 | 友だち追加時、定期的にキャッシュ更新 | UI 表示・運営者向けサポート |
| プロフィール画像 URL | 同上 | UI 表示（任意） |

LINE User ID は LINE プラットフォーム内部の識別子であり、それ単体では特定の個人を識別することはできません。

### 1.2 利用者が本サービス上で入力する情報

| 情報 | 用途 |
|---|---|
| 学科試験問題への解答 | 採点・進捗管理・苦手分野分析 |
| モード選択（仮免/本免） | 出題範囲の制御 |
| 通知設定 | リマインド配信のオプトイン管理 |

### 1.3 自動的に記録される情報

| 情報 | 用途 |
|---|---|
| 出題履歴・回答履歴・正答率 | 進捗管理・問題品質改善 |
| アクセス時刻・操作ログ | サービス品質改善・障害解析 |

---

## 2. 利用目的

取得した情報は以下の目的でのみ利用します。

1. 学科試験対策の問題出題・採点・解説提供
2. 利用者の学習進捗管理および苦手分野分析
3. 本サービスの品質改善・問題品質向上
4. 障害解析および不正利用防止
5. 利用者からの問合せ対応
6. 法令改正等に伴う問題内容の更新

---

## 3. 第三者提供

以下の場合を除き、取得した情報を第三者に提供することはありません。

### 3.1 業務委託先（クラウドサービス）

本サービスの運営に必要な範囲で、以下のクラウドサービス提供者に情報を取扱わせます。これらは個人情報保護法における「委託」に該当します。

| 提供先 | 提供範囲 | 用途 |
|---|---|---|
| Google Cloud Platform（Google LLC） | LINE User ID から派生した内部 UUID、出題履歴、回答ログ等 | データ保管・処理基盤（Cloud Run / Firestore / BigQuery / GCS） |
| Vertex AI 経由の Anthropic Claude / Google Gemini | 問題生成プロンプト、ユーザー回答（誤答時の解説生成のみ） | 問題生成・解説生成 |
| LINE 株式会社 | LINE Messaging API による通信 | LINE Bot 機能の提供 |

**重要**: 利用者が直接入力したテキスト（自由記述・問合せ等）は、必要に応じて Vertex AI 経由で Claude / Gemini に送信されます。送信先のサービスは入力内容を学習に使用しません（Vertex AI のデータガバナンス方針に従う）。

### 3.2 法令に基づく開示

法令に基づく開示請求があった場合、または人の生命・身体・財産の保護のために必要な場合は、関係機関に情報を開示することがあります。

---

## 4. データ保管期間

| データ種別 | 保管期間 | 削除タイミング |
|---|---|---|
| ユーザープロフィール | 利用継続中 + 30 日 | LINE ブロック検知から 30 日経過後に物理削除 |
| 出題履歴・回答ログ | 12 ヶ月（分析目的） | 12 ヶ月経過後に統計化（個人特定不能化） |
| LINE 表示名・画像のキャッシュ | 7 日間 | TTL 経過後に LINE API から再取得 |
| 削除リクエスト処理 | 即時論理削除 → 7 日後物理削除 | 「データを削除」コマンド実行時 |

---

## 5. 利用者の権利

利用者は以下の権利を有します。

### 5.1 データ削除リクエスト

LINE Bot に「データを削除」と送信することで、本サービスに保管された当該利用者の情報を削除できます。削除は即時に反映され、7 日後に完全に物理削除されます。

### 5.2 開示請求

利用者は、運営者に対し、本サービスが保有する自己の情報の開示を請求することができます。

### 5.3 LINE 友だち解除

LINE 公式アカウントの友だち解除（ブロック）を行った場合、検知から 30 日経過後に当該利用者の情報を物理削除します。

---

## 6. セキュリティ対策

1. すべての通信は TLS で暗号化されます。
2. データは Google Cloud Platform 上で暗号化保存されます。
3. 認証情報・API キー等の秘匿情報は Google Cloud Secret Manager で管理します。
4. 内部システム間の通信はワークロードアイデンティティ（Workload Identity Federation）により認証されます。
5. 開発・運用に従事する者は運営者本人のみであり、外部委託は行いません。

---

## 7. クッキー等の利用

本サービスは LINE プラットフォーム上で動作するため、ウェブブラウザのクッキーは使用しません。LINE Login（LIFF）連携を Phase 2 以降で導入する場合は、本ポリシーを更新します。

---

## 8. 子どもの個人情報

本サービスは運転免許取得を目指す方（原則 18 歳以上）を対象としています。利用者が 18 歳未満の場合は、保護者の同意のもとで利用してください。

---

## 9. 改定

本ポリシーは、必要に応じて改定することがあります。改定後の内容は、本サービス上に掲示された時点から効力を生じます。
